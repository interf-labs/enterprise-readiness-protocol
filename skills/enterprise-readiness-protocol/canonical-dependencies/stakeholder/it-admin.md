# IT Admin

Vendor needs an IT administrator to provision accounts, configure firewall rules, set up service accounts, or manage infrastructure-level access. The hands-on counterpart to the security team — IT admin executes while security approves.

## Template

- **what:** IT admin to provision service accounts and configure network access
- **ready:** Service accounts created and network rules applied

## Matches

Use this type when vendor says: "IT admin", "sys admin", "infrastructure team", "someone to provision accounts", "firewall configuration", "network access setup"
