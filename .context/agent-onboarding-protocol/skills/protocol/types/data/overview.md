# data — Data Access & Preparation

Vendor needs enterprise to provide, prepare, or grant access to specific data. Match when vendor says "we need your data", "data export", "map your fields", "sample data", "historical records".

## Types

| Canonical | One-liner | Detail |
|---|---|---|
| `data.historical-export` | Bulk export of historical data (specify timeframe) | [exports.md](./exports.md) |
| `data.real-time-feed` | Continuous real-time data stream or CDC | [exports.md](./exports.md) |
| `data.field-mapping` | Human-assisted mapping of custom fields to vendor schema | [mapping.md](./mapping.md) |
| `data.schema-access` | Understanding of enterprise data model and relationships | [mapping.md](./mapping.md) |
| `data.sample-dataset` | Sample/test data for validation | [access.md](./access.md) |
| `data.pii-access` | Access to personally identifiable information | [access.md](./access.md) |
