# infrastructure — Environments & Network

Vendor needs enterprise to provision environments, network access, or compute resources. Match when vendor says "staging environment", "test environment", "firewall", "allowlist", "network access".

## Types

| Canonical | One-liner | Detail |
|---|---|---|
| `infra.test-environment` | Staging/sandbox environment with representative data | [environments.md](./environments.md) |
| `infra.compute` | Compute resources for vendor workload | [environments.md](./environments.md) |
| `infra.network-access` | Firewall rules, IP allowlisting, VPN | [network.md](./network.md) |
| `infra.dns` | DNS records, custom domain setup | [network.md](./network.md) |
